# IT3040-ITPM Assignment 1: Automated Testing

This repository contains the Playwright automation scripts for Assignment 1 of the IT3040 (IT Project Management) module. The objective is to test the Singlish-to-Sinhala transliteration system (SwiftTranslator).

**Student ID:** IT23768758

## Prerequisites
- Node.js (v14 or higher)
- NPM (Node Package Manager)

## Installation
1. Navigate to the project directory:
   ```bash
   cd IT23768758
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Install Playwright browsers:
   ```bash
   npx playwright install
   ```

## Running Tests
To run all functional and UI tests:
```bash
npx playwright test
```

To view the HTML test report:
```bash
npx playwright show-report
```
